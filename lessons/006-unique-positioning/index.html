Manipulatin the z-index

The z-index defines the "level" of elements, that are layered on top of each other. This property can only be used for elements that have the "position" value "relative" or "absolute".

The value should be an integer. With a positive value it will add levels and with a negative value you can tell the element to go backwords the given amount of levels.
