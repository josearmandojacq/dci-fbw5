Background Size

With Internet Explorer versions > 10 and all other major browsers in ther last 3 versions you'll be able to use a new CSS3 property "background-size".
The property accepts a few different values, including length and keyword values.

When using length values, we can specify a width and a height value by using two space-separated values.
The first value will set the width of the background image, while the second value will set the height of the background image.
It’s important to note that percentage values are in relation to the element’s size, not the background image’s original size.

You can use the keyword "auto" for one of the two values, to keep the aspect ratio, when setting only the width or the height.

This element uses 50% of it's size for the background image, while
keeping the aspect ratio of the original image.

background-size: 50% auto;

In addition to length background-size property values, there are also cover and contain keyword values available to the background-size property.

The cover keyword value specifies that the background image will be resized to completely cover an element’s width and height.
The background image’s original aspect ratio will be preserved, yet the image will stretch or shrink as necessary to cover the entire element.

The contain keyword value, on the other hand, specifies that the background image will be resized to reside entirely contained within an element’s width and height.
In doing so the background image’s original aspect ratio will be preserved, but the image will stretch or shrink as necessary to remain within the width and height of the element.
