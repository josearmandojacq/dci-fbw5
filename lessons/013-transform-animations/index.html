CSS Transforms & Animations

Transforms

Rotate

The "rotate" value can be used to rotate the element.

Positive value: clockwise
Negative value: counter clockwise

 

Scale

The "scale" value can be used to change the size of an element.

It's important to know, that the element will not change it's position, as you'll be able to see below.

Translate

The "translate" value can be used to change the position of an element, like we know from absolute and relative positioning.

 

Skew

The "skew" value can be used to distort elements.

 

Combined Transforms

You can can combine transforms, by using a space seperated list as value of the "transform" property.

 

Animations

There's are detailed articles about the animation property at Mozilla's Developer Network:

  • The animation property
  • Using the animation property

The animation property can be used to animate elements and use moving transformations, etc. that have only been possible with JavaScript in the old days.
