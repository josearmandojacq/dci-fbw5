Media Queries

Targeting media types

Printing stylesheets

@media print {
    p {
        font-size: 12pt;
    }
}

Screen stylesheets

@media screen {
    p {
        font-size: 12pt;
    }
}

Targeting multiple devices

@media screen, print {
    p {
        font-size: 12px;
    }
}

Targeting media features

Screen size

We can react on different aspects of the browser's screen size, viewport, etc. 

/* Show 2 columns instead of one on larger screens */
@media (min-width: 992px) {
    .container {
      width: 50%;
    }
}

Combining media features

/* The background color is changed, if the resolution is between 760 and 860 pixels.*/
@media (min-width: 760px) and (max-width: 860px) {
    .container-2 {
        background: #ffff00;
    }
}
